Definitions of Terms We Have Learned So Far

HTML
HTML (HyperText Markup Language) is the language that web pages are built upon, which are then "painted" onto the screen by the web browser.
CSS
CSS (Cascading Style Sheets) is the set of properties and values that describe how the web page should look like.
FTP
FTP (File Transfer Protocol) is the Internet protocol that transfers files between computers connected to the Internet.
SSH
SSH (Secure Shell) is a cryptographic network protocol for operating network services securely over an unsecured network.
